Plan for practical lessons & tasks for students.

1. Introduction to base Data Science & Data Mining Python libraries (NumPy, Pandas).
   1. Doing base operations with vectors & matrices in NumPy.
   2. Working with pd.Series (indexes, adding data and changing content).
   3. Working with pd.DataFrame (indexes, adding data, modifying data).
2. Data Normalizations and a bit visualizations.
   1. Load & visualize the data distributions.
   2. Do data transformations: (Z-norm Scaling, Min-Max Norm, Robust Scaling, and other)
3. Working with missing values.
   1. Prepare data with missing values (manually remove data)
   2. Try different approaches of filling the data 
   (back fill, front fill, mean fill, median fill, specific value fill, etc.)
4. Working with non-numeric data.
   1. Working with Datetime.
   2. Working with categorical data.
   3. Working with other string data.
5. Data visualizations.
   1. Advanced visualizations with Seaborn.
   2. Advances interactive visualization with Plotly.
   3. Producing animations with Matplotlib.
6. The full Data Scientists / Data Miner working pipeline for data processing.
   1. Loading the data.
   2. Analyzing, doing transformations and repeating.
   3. Data tests: z-score, quantiles, data skewness, homo/hetero- scedasticity.
   4. Useful visualizations.
   5. Correlation tests to the target variable.
   6. Removing redundant or noizy features.
7. Applying the pipeline on the actual data processing trying various ML methods
   (consider Linear Regression, Logistic Regression and Decision Tree).
